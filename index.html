6 Rules to Variable Naming

1:MUST begin W/ a letter, $, or_

2:NO Dashes or Periods

3:NO keywords/reserved words(ie:var, let, true..)

4:Variables are case sensitive, DON'T use same words with diff. cases

5:Use variables name that ACTUALLY describes that variable(firstName: persons first name

6:Use camelCase or underscore(firstName, lastName, my_Name, car_Model)

Built in Objects - Page 121

Browsers contain built in objects that contain basic functionality that's needed for basic scripts

Can be accessed using the dot(.) notation

Tool kit comes in three 'compartments'

1: Browser Object Model: Represents current window/tab

2: Document Object Model: Uses objects to create a representation of the current page

3: Global JS Objects: Represents things that JS needs to create a model of(i.e dates or times)
