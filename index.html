CHAPTER 7: JQUERY

jQuery helps developers accomplish a variety of common JS tasks quickly and consistently, majority browser support without fallback code needed(sounds good)

  • Select Elements: Simplified in Jquery(css-ish style)

  • Perform Tasks: Updates/animates/loops elements in the DOM tree in a single line of code

  • Handle Events: Allows attaching event listeners without having to write fallback code for older browsers

    • TWO Steps:

    1: FIND ELEMENTS USING CSS-STYLE SELECTOR

    Use jQuery() or $()

    EX: $('li.hot') will find all <li> with a class of 'hot'

    2: CHANGE ELEMENT USING JQUERY METHODS

    EX: $('li.hot').addClass('complete') will add 'complete' class to all <li> with hot class elements
